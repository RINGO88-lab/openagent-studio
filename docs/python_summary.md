## 一、学过的知识点

### 1. 基础语法（P01-P21）
- 变量、数据类型、输入输出
- 字符串定义、拼接、格式化
- 运算符（算术/赋值/比较/逻辑）

### 2. 流程控制（P22-P35）
- if / elif / else
- match 模式匹配
- while 循环
- for 循环 + range()
- break / continue
- 嵌套循环

### 3. 数据容器（P36-P56）
- List：增删改查、切片、推导式、排序
- Tuple：不可变、解包、组包
- String：split/join/strip/replace/格式化
- Set：交集/并集/差集、去重
- Dict：增删改查、遍历、推导式、get 安全取值
- 数据容器选型：什么场景用什么

### 4. 函数（P57-P70）
- 函数定义、参数、返回值
- 变量作用域（局部/全局）
- 传参方式、默认参数、不定长参数
- lambda 匿名函数
- 递归

### 5. 文件操作 + 异常（自学补充）
- open / with 语句
- 文件读取（read/readlines）
- 文件写入（w 模式 / a 模式）
- JSON 序列化（json.dump / json.load）
- try / except / finally
- 自定义异常

### 6. 类型注解 + 模块（P71-P76）
- 变量类型注解
- 函数参数/返回值注解
- import / from ... import
- 自定义模块
- 包（package）

### 7. 面向对象（P77-P85）
- 类和对象
- __init__ 构造函数
- self 的含义
- 实例方法、魔法方法
- 实例属性与类属性

### 8. 异常（P86-P87）
- try / except / else / finally
- 异常类型
